# Task 3 - Multi Vagrant Machine


### In order to use this VM:
`git clone git@github.com:victorsibanda/nginxrevproxy.git`


### To Run Use:

`vagrant up`


#### For app

- `vagrant ssh app`
- `cd /home/ubuntu/app`
- `sudo npm install` - Only if npm start fails
- `sudo npm start`

Go to  ---> https://development.local/
